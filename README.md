这是我的第一个 Python 项目！

## 项目内容

### 猜数字游戏 (game.py)

一个简单有趣的猜数字游戏。

**游戏规则：**
- 计算机随机生成 1-100 之间的一个数字
- 你需要猜出这个数字
- 每次猜测后，计算机会提示你的猜测是太大还是太小
- 猜对后，显示你用了多少次尝试
- 可以选择继续玩或退出

## 如何运行

### 前提条件
- 安装 Python 3.x（[下载地址](https://www.python.org/)）

### 运行游戏

在终端/命令行中运行：

```bash
python game.py
```
